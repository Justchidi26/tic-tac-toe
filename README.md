# Tic Tac Toe (React)

A simple and interactive Tic Tac Toe game built with **React**. This project demonstrates core React concepts like components, state management, and event handling.

---

## Features

* Two-player gameplay (X and O)
* Winner detection logic
* Game state updates in real-time
* Reset / Restart game functionality
* Clean and responsive UI

---

## Tech Stack

* React (Functional Components)
* JavaScript (ES6+)
* CSS

## Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Justchidi26/tic-tac-toe.git
```

2. Navigate into the project folder:

```bash
cd tic-tac-toe
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
# or
npm start
```

5. Open in browser:

```
http://localhost:3000
```

---

## How It Works

* The board is made up of 9 squares.
* Players take turns clicking squares to place **X** or **O**.
* After every move, the app checks for a winning combination.
* If a player wins, the game displays the winner.
* If all squares are filled with no winner, the game ends in a draw.
